# Tejo Voice Assistant

This project is a simple voice assistant that listens for commands and performs various tasks such as playing music, telling jokes, generating QR codes, and more. It uses multiple Python libraries to handle voice recognition, text-to-speech, and other functionalities.

## Requirements

Before running the script, you need to install the following dependencies:

```bash
pip install speechrecognition pyttsx3 pywhatkit wikipedia pyjokes qrcode transformers
```

## How to Run

1. Download the necessary files.
2. After installing the required dependencies, run the script with the following command:

```bash
python index.py
```

## Voice Commands

Once the script is running, it will listen for the keyword **"Tejo"**. After detecting the keyword, you can use the following voice commands:

- **Play a song on YouTube**  
  Say: `Tejo, play [song name]`

- **Get the current time**  
  Say: `Tejo, what is the time`

- **Tell a joke**  
  Say: `Tejo, tell me a joke`

- **Generate a QR code**  
  Say: `Tejo, generate QR code`

- **Activate the AI model**  
  Say: `Tejo, Ask`

Make sure to speak clearly and follow the correct command format for the assistant to understand and perform the task correctly.
